1 - O que é HTML?

O html significa HyperText Markup Language e como o nome indica é uma linguagem de marcação. É usada para estruturar o conteúdo de páginas web. É só o "esqueleto" uma vez que para efeitos visuais usamos CSS.

2 - O que é um elemento HTML? Dê exemplos.

Um elemento é uma maneira de dizer conteúdo numa página. Por exemplo um <p> Lorem </p>é um elemento que possui tag de abertura, tag de fechamento e conteúdo. Existem tags que nao requerem fecho como <hr> e <br>, que também são elementos. Outros exemplos : <img src="#" alt= "#">; <span> Lorem </span> e <div> Lorem </div>.

3 - Que ferramentas são necessárias para desenvolver um site HTML?

Um editor de texto (o próprio notepad do windows lê ficheiros .html .css e até .js) e um browser web para visualização dos resultados. Recomendo no entanto o uso de um editor de código como o VS code, mas o notepad faz a função.

4 - Quais são os elementos que compõem a estrutura base do HMTL?
Além da estrutura base, o que mais devemos acrescentar para seguirmos as boas práticas de programação?

A estrutura base do HTML inclui o <!DOCTYPE html>, as tags <html>, <head> <title> </title> </head>, <body> </body> e o fechamento da tag no final </html>. Além desta estrutra, devemos aplicar o atributo lang na tag html <html lang="pt"> para definir corretamente a linguagem do site, o <meta charset="UTF-8" relativamente aos caractéres usados, e para responsividade em dispositivos móveis, devemos usar o <meta name="Viewport" content= "width=device-width, initial-scale=1.0">. Podemos e devemos ainda usar tags de acessibilidade como o <alt>. Como boas práticas deve ser levada a identação de todo o projeto com cuidado e rigor.

5 - O elemento <br> necessita de ser fechado?

Uma vez que se trata de uma tag sem fechamento, o elemento <br>, não precisam de ser fechadas porque não possuem conteúdo e interno e executam o seu proposito só com a sua presença sem fechamento. A mesma se coisa se aplica a tags como <hr> ou <img>.

6 - Para que servem os atributos? Dê um exemplo.

Os Atributos servem para adicionar informações ou por outras palavras, dar propriedades ás tags ou elementos. Por exemplo, a tag <img> precisa dos atributos src= e alt= para estar devida e corretamente preenchida.

7 - O que é a indentação e para quê ela serve?
Do ponto de vista do código HTML, a indentação é obrigatória para o seu funcionamento?

A indentação é o espaço horizontal usado no inicio das linhas de código, para mostrar a hierarquia entre os elementos. Ajuda a organizar visualmente o código, o que facilita a escalação do código para outros programadores, porque ajuda a entender a estrutura do código.

Relativamente ao HTML, os navegadores não têm em consideração o espaçamento ou quebras de linha, por isso o código iria correr sem erros de maior. No entanto, pelas razões acima mencionadas, torna-se essencial ter o código bem indentado.
